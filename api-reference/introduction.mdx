## API Overview

CodinIT provides two distinct API versions:

1. **Web API** (`openapi.json`) - Production-ready API for the hosted CodinIT platform at codingit.vercel.app
2. **Local API** (`openapi-schema.json`) - Development API for local CodinIT instances

Both APIs are documented using OpenAPI 3.x specifications and support comprehensive AI-powered coding workflows, sandbox execution, GitHub integration, and deployment capabilities.

## Authentication

Both API versions support multiple authentication methods depending on the endpoint:

**Web API Authentication:**
- Bearer tokens (JWT)
- Supabase session tokens

**Local API Authentication:**
- Cookie-based API keys
- Bearer tokens

```json Web API
"security": [
  {
    "BearerAuth": []
  },
  {
    "SupabaseAuth": []
  }
]
```

```json Local API
"security": [
  {
    "CookieAuth": []
  },
  {
    "BearerAuth": []
  }
]
