### Overview

Netlify provides a powerful platform for deploying modern web applications with continuous deployment, global CDN, and serverless functions. CodinIT's Netlify integration offers complete deployment automation and management.

### Setup Instructions

<Steps>
  <Step title="Create Netlify Account">
    Visit [Netlify](https://netlify.com) and create your account if you haven't already.
  </Step>

  <Step title="Generate Personal Access Token">
    Go to **User Settings > Applications** and click **"Generate new token"**.
  </Step>

  <Step title="Configure in CodinIT">
    Navigate to **Settings > Deployments** and add your Netlify token.
  </Step>

  <Step title="Connect Git Repository (Optional)">
    Link your GitHub/GitLab repository for automatic deployments.
  </Step>

  <Step title="Deploy Your Site">
    Click **"Deploy to Netlify"** to publish your application instantly.
  </Step>
</Steps>

### Configuration Options

<ParamField name="NETLIFY_ACCESS_TOKEN" type="string" required>
  Your Netlify personal access token. Get this from [Netlify Applications](https://app.netlify.com/user/applications).
</ParamField>

<ParamField name="NETLIFY_SITE_NAME" type="string">
  Custom site name for new deployments. If not provided, a unique name will be generated.
</ParamField>
